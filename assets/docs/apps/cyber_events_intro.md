# An interactive look at 15,000+ publicly reported cyber events, 2014–2025

This project is an interactive Tableau dashboard exploring the cyber threat landscape over the past twelve years: who the attackers are, what motivates them, which industries and countries they target, and how those patterns have shifted over time. It began as my term project for CSIS 3860 (Data Visualization) at Douglas College, and I have since cleaned it up and republished it here as a portfolio piece.
